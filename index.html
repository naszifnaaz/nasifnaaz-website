Latest Projects.

Here you can find some of the projects that I created recently

Facial Emotion Recognition

FER is a project that make use of Machine Learning techniques and Computer vison to accurately determine the emotion of users in real time using a camera module. The projects application in mood based music system is also considered.

OpenMW Mod Manager

OpenMW has no GUI for mod directory installation and management. So i created Mod Manager plugin to export your VFS, plugin selection and load order to OpenMW.

Online Library Portal

An interactive web portal for automating various manual processes done by librarian. The system keep track of all the actions related to library using a GUI based web application.

Railway Management System

This is a Java based standalone application built using javafx framework and Mysql.The Railway Reservation System facilitates the admin to enquire about the trains available on the basis of source and destination, Booking and Cancellation of tickets, Creating train details and its routes.

Flappy Bird Clone

Flappy Bird Clone is my version of the iconic flappy bird game which took the internet by storm in 2013. This project is created using the pygame module in Python with the use of basic animation using sprites. It is a desktop only verion which is supposed to be played with a mouse.

Sudoku Solver

Sudoku (数独) is a logic-based, combinatorial number-placement puzzle.This is a Sudoku game created using pygame module in Python. It also includes a sudoku solver which solves the puzzle for you using the backtracking algorithm.
